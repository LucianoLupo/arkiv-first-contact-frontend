Decentralized Indexing Platform

How DecentralIndex Works

1. Listen

DecentralIndex listeners monitor smart contracts (Uniswap, Aave) for specific events.

2. Store

Events are normalized and stored on the Arkiv DB-chain with queryable attributes.

3. Query

Dapps query DecentralIndex directly via SQL-like logic. No API keys, no central server.
